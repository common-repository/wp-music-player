== Description ==

WP Music Player plugin allows visitor to Play music file on Wordpress Post page.

WP Music Player has number of options to configure.

You can change configuration setting for music player.

In admin side you can Upload Music File And after that you can assign this file to Post

You can also  use this plugin for Song websites. You can describe song with the post and assign file

This Plugin use for Audio testimonial.

You can change Place of Audio Player By changing in Wordpress Hook.

By using `[music-player]` short code  in Post or page for display Music Player

For detailed documentation visit http://www.mitulhirpara.com/wp-music-player/

== Installation ==

1. Upload to the "wp-music-player" directory to `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Configure plugin
